About Daisyworld

This is an interactive experience where users can connect and create together.

Each participant becomes a bird with their own unique sound and color. Together, you create a song that exists only in this shared moment.

How to Use:

  • Select white or black using the color buttons
  • Click anywhere to create sprites
  • Watch them interact with the environment
  • Type messages to communicate with others

Created with p5.js and WebRTC
